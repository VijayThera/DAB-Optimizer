# Project Notes

## Setup
```commandline
python3 -m venv .venv
source .venv/bin/activate
pip install -e ~/lea_python_toolbox/
pip install dotmap
pip install tqdm
```

When pyCharm can for some reason not install the requirements, then do this in a shell:
```commandline
source .venv/bin/activate
pip install -r requirements.txt
```
